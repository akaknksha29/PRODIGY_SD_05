# Web Scraper (Task 05)

This project is a Java-based web scraper that extracts the latest headlines from a website using the jsoup library.

## Features

* Connects to a live website
* Fetches HTML content
* Extracts top 5 headlines
* Displays structured output

## Tech Used

* Java
* jsoup library

## Output Example

1. Headline 1
2. Headline 2
3. Headline 3

## Learning Outcome

Learned how to fetch and parse HTML data using Java and external libraries.
